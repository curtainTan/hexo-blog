问题来了

有这样一串字符串：

<p>你好，</p><p>world</p>

我要取出标签内的文字：你好，world

问题分析

我们要做的是把文字提取出来，怎么提取呢？
和正则相关的方法总共有6个：

  1. String#search
  2. String#split
  3. String#match
  4. String#replace
  5. RegExp#test
  6. RegExp#exec

每个方法的具体使用与需要注意的点可以看我之前的文章：正则小记。

这6个方法里面，只有match、exec和replace能将字符串提取出来，这里我们选用match和exec这两个api来实现。

提取字符串的方法选好了，下面就可以开心的写正则了。
